# Documentation AccessGuard

Ce dossier regroupe tous les documents produits pendant le projet AccessGuard.

## Organisation prévue

```text
docs/
├── architecture-reseau/  # Schémas réseau, VLAN, plan d’adressage et règles de sécurité
├── diagrammes/           # Cas d’utilisation, diagrammes d’activité, séquence et classes
├── maquettes/            # Maquettes des interfaces utilisateur
├── rapports/             # Rapports des rendus n°02 à n°06
└── README.md
```

## Rendu n°02 — Conception

Les documents de conception incluront :

* le contexte détaillé d’AsteriaTech ;
* les objectifs et le périmètre du projet ;
* les acteurs et cas d’utilisation ;
* les scénarios détaillés ;
* les diagrammes UML ;
* le modèle de données ;
* les maquettes ;
* l’architecture réseau ;
* le plan d’adressage ;
* les règles de sécurité ;
* le backlog, les epics et les sprints.

## Rendus suivants

Les prochains rendus présenteront progressivement :

* la mise en œuvre de l’application ;
* les tests ;
* la sécurité ;
* l’infrastructure Docker et réseau ;
* la pipeline DevSecOps ;
* le monitoring ;
* la démonstration finale.
