## Règles

- masquer tout token JWT ;
- ne jamais afficher de mot de passe ;
- utiliser des justifications professionnelles ;
- afficher le rôle connecté lorsque cela apporte une preuve RBAC ;
- conserver une résolution permettant de lire les textes.
